# BookReader Network Project - NT106

## 📋 Yêu cầu hệ thống
- Windows 10/11
- .NET 8.0 SDK
- Visual Studio 2022 (khuyến nghị)

## 🚀 Cách chạy:

### Demo cơ bản:
```
setup_and_run.bat
```

### Demo Load Balancing:
```
demo_load_balancing.bat
```

### Demo Internet (cần ngrok):
```
demo_internet.bat
```

## ✅ Tính năng đã hoàn thành:
- Socket programming với TCP
- Multi-client real-time communication
- Load balancing với round-robin
- SQLite database integration
- Google Drive integration
- Real-time chat system
